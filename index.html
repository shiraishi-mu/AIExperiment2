Pythonによる人工知能プログラミング実験

Author

広島市立大学情報科学部知能工学科編

Published

June 1, 2026

Abstract

本書は、広島市立大学情報科学部知能工学科の「知能工学実験Ⅱ」（人工知能プログラミング実験）の教材として作成されたものである。 Pythonによるオセロゲームの実装を通じて、ゲーム木探索、評価関数の設計、テスト駆動開発、バージョン管理などの人工知能プログラミングとシステム開発の基礎を学ぶことを目的とする。 本資料は2025年度まで使用された人工知能プログラミング実験のテキスト（LISP版）をもとに、2026年度の内容に合わせて改訂したものであり、説明の整理と最新のPythonの機能を活用したコード例の更新を行っている。 LISPからPythonへの移行にはChatGPTを補助的に活用した。

1 はじめに

人工知能の研究は、推論、探索、知識表現、機械学習、深層学習、生成AIへと発展してきた。 近年の生成AIの著しい性能向上は、大規模なデータと計算資源、そして適切なアルゴリズムが組み合わさった結果である。 このような華やかな成果の裏側には、人工知能の基礎的な概念である「状態をどのように表現するか」「限られた計算資源の中でどのように解を探索するか」という古典的な問いがある。 これらは、現在の機械学習やAIエージェントにも共通する考え方であり、人工知能を学ぶ上で避けて通れないテーマである。

本実験では、Pythonを用いてオセロゲームを段階的に実装することを通して、これらの基礎を体験的に学ぶ。

1.1 本実験の特徴

旧教材（2025年度まで）ではプログラミング言語としてLISPを用いていた。 LISPは記号処理に特化した古典的な人工知能言語であり、ゲーム木探索や知識表現の研究で長く使われてきた。 しかし、近年の機械学習・深層学習・生成AI研究は、ライブラリと開発エコシステムの豊かさからPythonが事実上の標準言語となっている。

LISPは、人工知能の黎明期から使われてきた歴史的な言語であり、シンプルな構文と強力なマクロ機能を持つ。 その抽象性の高さは、ラムダ計算や関数型プログラミング言語としての側面を持ち、現代でも重要な言語の一つである。

そこで2026年度からは、本実験もPythonによる実装に切り替える。 あわせて、現代のソフトウェア開発で必須となっている

  • Gitによるバージョン管理
  • pytest を用いた単体テスト
  • VSCode + Dev Container による再現可能な開発環境

を実験のフレームワークとして組み込む。

13 対戦実験とGUI
14 最終課題と評価

これらは大きく次の3つの段階に分かれている。

  1. 実装の基礎（1〜8章）：環境構築、ルール実装、minimax/αβ までを完成させる。
  2. 評価関数の設計（9〜10章）：人間の知識を評価関数として記述し、手動で改良する。
  3. 自動化の階層（11〜12章）：人間の手を離れて、最適化アルゴリズムや機械学習で評価関数を改良する。

11章と12章は、AI 研究の歴史でいえば「人間の知識を作り込む段階」から「最適化が知識を獲得する段階」、そして「データから学ぶ段階」への移行に対応している。 これは、本書の第1章で振り返る AI の歴史的発展（推論 → 知識ベース → 機械学習）の縮図でもある。

1.4 本書の使い方

各章には例題と練習がある。 例題には基準となる実装例が示されているので、まずは写経し、動作を確認してから先に進む。 練習は、その章で学んだ関数を自分の手で実装する課題である。 どの練習にも動作確認用の assert 文がついているので、これが通ることを確認しながら進めること。 こまめに git commit し、テストが通った状態を残すこと。
